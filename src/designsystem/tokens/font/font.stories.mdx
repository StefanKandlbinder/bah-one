# Design Tokens
Design tokens are a tech-agnostic way to store variables. We use tokens instead of hard coded values to ensure a better UI consistency across different platforms.

## Font
Text is interface, which is why weâ€™ve spent a great amount of time fine-tuning our typographic choices. Use the below tokens to set font-families and font-weights.
